# Project

A React-based user Directory Dashboard built as part of the BuyerForeSight Frontend Engineer Assessment

## Live Demo

## Features

### Dashboard

- Display users in a table or grid
- Show the following fields:
  - Name
  - Email
  - Phone
  - Company

### Search & sort

- Search: Filter users by name or email (client-side)
- Sorting: Sort users by Name and Company (ascending/descending)
- Note: didnt used debounce as it is filetring from fetched list and list is about 10 users

### User Detail Page

- Clicking a user row navigates to a detail view
- Display full user information

## API used

https://jsonplaceholder.typicode.com/users

## Tech stack

- React (vite)
- React Router Dom
- Tailwind CSS
- Fetch API
